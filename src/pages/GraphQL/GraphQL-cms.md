# 使用GraphQL-cms建立筆記存放位置
1. 建立資料格式(Schema)
2. 添加Models
    - 設定名稱
    - 添加 Fields (API的資料名稱) (可選擇是否為必填項目)
3. 組合Model
    - 在主要model上，選取Refence關聯指定Model (可選擇單對單 or 多對多)

## 建立content
1. 進入Content
    - 在主要moodel中，建立content。


## 設定API
1. Project settings > Api Access > Public Content API > Add permission(設定權限)
