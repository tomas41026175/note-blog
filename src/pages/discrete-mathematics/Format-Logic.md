# 形式邏輯 (Format Logic)
- 用語
    - statement = proposition = 命題(非A即B)
    - conjunction = and = ⋀ = ||
    - Disjunction = or = ⋁ = &&
    - Implication = 若A則B = →
    - Equivalence = ⇔ = A→B B→A
    - Negation = ' = 否定(not)
    - Tautologies = 恆真式 = 無論如何都為true
    - parentheses = 括號
    - syntax = 語法
 
- 重點 → ← ⇔ ⋀ ⋁
    1. (A→B)⇔(B'→A')⇔(A'⋁B)
    2. De Morgan's Laws
        *(A⋁B)'⇔A'⋀B' , (A⋀B)'⇔A'⋁B' 
    3. (A→B)'⇔A⋀B'
    4. (A⇔B)⇔[(A→B)⋀(B→A)]
    ::: danger
    5.Tautologies is a statement that is always true , no matter what
    :::
